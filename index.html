Flexbox is used for one-dimensional layouts, such as creating a navigation bar or aligning items in a row or column. It provides a simple way to align elements along a single axis and distribute space between them.

Grid is used for two-dimensional layouts, such as creating a grid of items or creating a layout with multiple rows and columns. It provides a powerful way to align elements along both the horizontal and vertical axes, and to create complex and responsive layouts.

The basic syntax of a media query is to include a media type.

Media queries are a feature of CSS that allows you to apply different styles to a web page based on the characteristics of the device or screen it's being viewed on. They allow you to create responsive designs that adapt to different screen sizes, resolutions, and orientations.
By using media queries, you can create a single HTML and CSS codebase that adapts to different devices and screen sizes, making your website more accessible and user-friendly across different platforms and devices.

The CSS box model is the way that CSS calculates the size and dimensions of elements on a web page.

The box model works by treating each HTML element as a rectangular box, with content, padding, borders, and margins. The content of the box is the area where the element's actual content, such as text or images, is displayed. The padding is the space between the content and the border, and the border is the edge of the box that surrounds the content and padding & the margin is the space outside of the border.

Semantic tags are HTML tags that describe the meaning or purpose of the content they contain.

They provide additional information about the structure and organization of the content, which can be used by browsers, search engines, and assistive technologies to understand the context and importance of the content.
For example, the .header tag is a semantic tag that is used to indicate the header of a document or section, the .nav tag is used to indicate navigation links, the .article tag is used to indicate a self-contained piece of content, such as a blog post or a news story, the .footer tag is used to indicate the footer of a document or section.
